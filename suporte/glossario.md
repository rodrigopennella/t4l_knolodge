# Glossário de Termos Técnicos

Termos utilizados no suporte técnico da T4L.

---

| Termo | Significado | Contexto de Uso |
|---|---|---|
| **AnyDesk** | Software de acesso remoto | O suporte acessa o computador do cliente para diagnóstico e correções |
| **Backup** | Cópia de segurança do banco de dados | Usado para restaurar o sistema em caso de falha grave |
| **Banco de Dados** | Base de dados do SAG (MySQL) | Armazena todos os dados do sistema — fica no servidor |
| **CF-e** | Cupom Fiscal Eletrônico | Documento emitido pelo equipamento SAT (usado em SP) |
| **CFOP** | Código Fiscal de Operações e Prestações | Classifica o tipo de operação fiscal — consultar contabilidade |
| **Certificado Digital** | Arquivo de autenticação eletrônica da empresa | Necessário para emissão de NFC-e; instalado no servidor |
| **Comanda** | Registro de pedido vinculado a mesa ou cliente | Comum em restaurantes e bares |
| **CPF / CNPJ** | Identificação de pessoa física / jurídica | Solicitado na venda para emissão de nota fiscal |
| **CRT** | Código de Regime Tributário | Define o enquadramento fiscal da empresa (Simples, Normal, MEI) |
| **CST** | Código de Situação Tributária | Classificação fiscal do produto — obrigatório para NFC-e |
| **DANFE** | Documento Auxiliar da Nota Fiscal Eletrônica | Representação impressa da NF-e |
| **iFood** | Plataforma de delivery integrada ao SAG | Pedidos feitos no iFood chegam automaticamente ao SAG |
| **IP Fixo** | Endereço de rede permanente | Recomendado para o servidor para evitar perda de conexão |
| **NFC-e** | Nota Fiscal de Consumidor Eletrônica | Emitida diretamente pelo SAG para o consumidor final |
| **NF-e** | Nota Fiscal Eletrônica | Emitida para operações entre empresas (B2B) |
| **Nobreak** | Equipamento de energia ininterrupta (UPS) | Protege o servidor de quedas de energia |
| **Pinpad** | Terminal de pagamento com cartão | Conectado via USB ao caixa — realiza transações TEF |
| **Range de IP** | Faixa de endereços de rede | Caixas e servidor devem estar no mesmo range para se comunicar |
| **Roteador** | Equipamento que distribui a rede Wi-Fi e cabeada | Ponto central da rede do estabelecimento |
| **SAG** | Sistema de gestão para food service | Sistema PDV da T4L |
| **SAT** | Sistema Autenticador e Transmissor | Equipamento fiscal homologado pela SEFAZ (SP) |
| **SEFAZ** | Secretaria da Fazenda Estadual | Órgão que autoriza documentos fiscais eletrônicos |
| **Servidor** | Computador central que hospeda o banco de dados | Todos os caixas e tablets se conectam a ele |
| **Sincronização** | Atualização de dados entre sistemas | Tablet, caixa e servidor precisam estar sincronizados |
| **Switch** | Equipamento de rede para múltiplas conexões cabeadas | Conecta vários dispositivos à mesma rede local |
| **Tabela de Preço** | Configuração de preços por contexto | Ex: preço diferente para balcão e delivery |
| **TEF** | Transferência Eletrônica de Fundos | Sistema de pagamento com cartão integrado ao caixa |
| **Totem** | Terminal de auto-atendimento | O cliente faz o pedido e paga sem atendente |
| **Validar Online** | Sincronizar licença com servidor de ativação | Feito quando o SAG solicita revalidação da licença |
| **VR / VA** | Vale Refeição / Vale Alimentação | Formas de pagamento com benefícios de alimentação |
| **XML** | Arquivo de dados fiscal eletrônico | Gerado automaticamente para cada documento fiscal emitido |
