# SAG Web — Painel de Gestão Online

O **SAG Web** é um aplicativo web que oferece acesso remoto às informações do SAG do estabelecimento — relatórios financeiros, vendas, estoque e outros dados — sem precisar estar na frente do computador do estabelecimento.

Acesso: https://sag.t4l.com.br

---

## O que é

- Aplicativo 100% web (acessa pelo navegador, sem instalação)
- Vínculo direto com o banco de dados do cliente
- Permite visualizar informações de vendas, financeiro e estoque em tempo real
- Voltado para o gestor/dono acompanhar o negócio de qualquer lugar

## Configuração

A configuração da conta é feita pela equipe de suporte da T4L. O cliente não configura por conta própria — se não tiver acesso, deve solicitar ao suporte.

---

## Primeiro Acesso

1. Acesse **sag.t4l.com.br**
2. Clique em **Primeiro Acesso**
3. Informe o **e-mail cadastrado**
4. Defina a **senha**
5. O sistema já avança para o dashboard

> A partir do segundo acesso, basta informar e-mail e senha normalmente.

---

## Funcionalidades

O SAG Web permite visualizar:

- **Dashboard** — visão geral do faturamento e indicadores do negócio
- **Relatórios financeiros** — receitas, despesas, fluxo de caixa
- **Relatórios de vendas** — faturamento por período, produtos mais vendidos, ticket médio
- **Estoque** — posição atual de estoque e movimentações

> O SAG Web é somente leitura — não é possível lançar ou alterar dados pelo aplicativo, apenas consultar.
